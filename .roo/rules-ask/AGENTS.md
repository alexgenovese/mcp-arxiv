# Project Documentation Rules (Non-Obvious Only)

- **arXiv Feed Categories**: Published categories start with `cs.` (e.g., `cs.AI`, `cs.LG`) and `physics.`; use RSS endpoint `https://rss.arxiv.org/rss/{category}` for Atom format.
- **Missing implementation detail**: The `arxiv_read_abstract_html` tool is referenced in README as a potential future implementation but does not exist in current codebase.
